## Getting Started

### Prerequisites

- Java 17 or newer (or Java version compatible with your project)
- [Maven](https://maven.apache.org/) or [Gradle](https://gradle.org/) (based on your build system)
- Git

### Running the Application

1. **Clone the repository:**
   ```bash
   git clone https://github.com/kaemiks/Spring-Boot-Project.git
   cd Spring-Boot-Project
   ```

2. **Build and run (Maven example):**
   ```bash
   ./mvnw spring-boot:run
   ```

   _or_

   **Package and run:**
   ```bash
   ./mvnw clean package
   java -jar target/*.jar
   ```

3. **Access the application:**
   Usually runs on [http://localhost:8080](http://localhost:8080)

### Configuration

- Application properties can be set in `src/main/resources/application.properties`.
